# hexColor

Creates a pipeline validation action that validates a hexColor.

````ts
const Validation = hexColor<TInput>(message);

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `hexColor` you can validate the formatting of a string. If the input is not a hex color, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />
````
