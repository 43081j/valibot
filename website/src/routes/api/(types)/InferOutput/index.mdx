# InferOutput

Infer output type.

```ts
// Create object schema
const ObjectSchema = object({
  key: pipe(
    string(),
    transform((input) => input.length)
  ),
});

// Infer object output type
type ObjectOutput = InferOutput<typeof ObjectSchema>; // { key: number }
```
