# string

Creates a string schema.

```ts
// String schema with pipe
const schema = string(pipe);

// String schema with message and pipe
const schema = string(message, pipe);
```

## Parameter

- `errorMessage` <Property {...properties.errorMessage} />
- `pipe` <Property {...properties.pipe}/>

## Return

- `schema` <Property {...properties.stringSchema} />

## Examples

```ts
// Schema to validate an email
const emailSchema = string([
  minLength(1, 'Please enter your email.'),
  email('The email is badly formatted.'),
  maxLength(30, 'Your email is too long.'),
]);

// Schema to validate a password
const passwordSchema = string([
  minLength(4, 'Your password is too weak.'),
  maxLength(30, 'Your password is too long.'),
  regex(/^[a-zA-Z0-9]/, 'Your password must contain only letters and numbers.'),
]);

// Schema to validate a URL
const urlSchema = string([url('The URL is badly formatted.')]);
```
