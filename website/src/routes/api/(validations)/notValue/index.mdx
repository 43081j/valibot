# notValue

Verify tat a value is different from the given value.

## Failing Example

```ts
const schema = z.string([notValue('foo')]);
const result = schema.safeParse('foo'); // Error
```

## Passing Example

```ts
const schema = z.string([notValue('foo')]);
const result = schema.safeParse('bar'); // Succeeds
```
