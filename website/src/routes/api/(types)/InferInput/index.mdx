# InferInput

Infer input type.

```ts
// Create object schema
const ObjectSchema = object({
  key: pipe(
    string(),
    transform((input) => input.length)
  ),
});

// Infer object input type
type ObjectInput = InferInput<typeof ObjectSchema>; // { key: string }
```
