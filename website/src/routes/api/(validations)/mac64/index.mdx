# mac64

Creates a pipeline validation action that validates a mac 64 bit address. If you want to validate both 48 bit mac address and 64 bit address, you can use `mac` validation action and if you want
to validate the 48 bit mac address, you can use `mac48` validation action.

```ts
const Validation = mac64<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `mac64` you can validate the formatting of a string. If the input is not a mac 64 bit address, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `mac64` can be used.

### Mac 64 bit address schema

Schema to validate a mac 64 bit address.

```ts
const Mac64AddressSchema = string([
  mac64('The mac address is badly formatted.'),
]);
```
