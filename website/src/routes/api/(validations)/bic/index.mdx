# bic

Creates a pipeline validation action that validates BIC (ISO 9362 Business Identifier Code).

```ts
const Validation = bic<TInput>(message);
```

## Generics

- `TInput` <Property {...properties.TInput} />

## Parameters

- `message` <Property {...properties.message} />

### Explanation

With `bic` you can validate the formatting of a string, if it's valid BIC - Business Identifier Code. If the input is not a bic, you can use `message` to customize the error message.

## Returns

- `Validation` <Property {...properties.Validation} />

## Examples

The following examples show how `bic` can be used.

Schema to validate an email.

```ts
const BicSchema = string([
  minLength(8, 'Please enter a BIC.'),
  bic('Input is not a valid BIC .'),
  maxLength(11, 'Your BIC is too long.'),
]);
```
