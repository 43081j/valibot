# arrayAsync

Creates an array async schema.

```ts
const SchemaAsync = v.arrayAsync<TItem, TMessage>(item, message);
```

## Generics

- `TItem` <Property {...properties.TItem} />
- `TMessage` <Property {...properties.TMessage} />

## Parameters

- `item` <Property {...properties.item} />
- `message` <Property {...properties.message} />

### Explanation

With `arrayAsync` you can validate the data type of the input asynchronously. If the input is not an array, you can use `message` to customize the error message.

> If your array has a fixed length, consider using <Link href="../tupleAsync/">`tupleAsync`</Link> for a more precise typing.

## Returns

- `Schema` <Property {...properties.Schema} />

## Examples

The following examples show how `arrayAsync` can be used.

### String array async schema

Schema to validate an array of strings asynchronously.

```ts
const StringArraySchemaAsync = v.arrayAsync(
  v.string(),
  'An array is required.'
);
```

### Object array async schema

Schema to validate an array of objects asynchronously.

```ts
const ObjectArraySchemaAsync = v.arrayAsync(v.object({ key: v.string() }));
```

### Validate length

Schema that validates the length of an array asynchronously.

```ts
const ArrayLengthSchemaAsync = v.pipeAsync(
  v.arrayAsync(v.number()),
  v.minLength(1),
  v.maxLength(3)
);
```

### Validate content

Schema that validates the content of an array asynchronously.

```ts
const ArrayContentSchemaAsync = v.pipeAsync(
  v.arrayAsync(v.string()),
  v.includes('foo'),
  v.excludes('bar')
);
```
