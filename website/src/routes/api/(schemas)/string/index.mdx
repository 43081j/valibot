# string

Creates a string schema.

```ts
// String schema with pipe
const schema = string(pipe);

// String schema with message and pipe
const schema = string(message, pipe);
```

## Parameter

- `errorMessage` <Property type='string'/> = 'Invalid type'
- `pipe` <Property type={{ type: 'custom', name: 'Pipe<string>' }}/> = []

## Return

- `schema` <Property type={{ type: 'custom', name: 'StringSchema' }}/>

## Examples

```ts
const schema = string();
parse(schema, 'string'); // returns 'string'
parse(schema, 123); // throw ValiError 'Invalid Type'

const schemaWithCustomErrMsg = string('custom error message');
parse(schemaWithCustomErrMsg, 123); // throw ValiError 'custom error message'

string([maxLength(5)]);
string([minLength(5)]);
string([length(5)]);

string([ulid()]);
string([regex(regex)]);
string([includes(string)]);
string([startsWith(string)]);
string([endsWith(string)]);
string([datetime()]);
string([ip()]);
```
